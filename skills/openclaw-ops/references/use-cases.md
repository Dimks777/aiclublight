# Кейсы по типу эксперта

> Что делать в первую очередь в зависимости от ниши и задач.

---

## Психолог / Коуч / Терапевт

**Старт:**
1. `начать` → интервью
2. `распаковка личности` → уникальный почерк
3. `threads 5` → первые посты

**Основные скиллы:** threads (охваты), storytelling (кейсы клиентов), telegram (глубокий контент), carousel (обучающие), reels (советы)

**Формулы:** "История клиента" → доверие. "Развенчание мифа" → охваты. "Пошаговая инструкция" → сохранения.

**Не делать:** Сухие научные тексты без историй. Длинные посты без хуков.

---

## Нутрициолог / Фитнес-тренер / Врач

**Старт:** `начать` → `threads 5` (мифы и разоблачения)

**Основные скиллы:** threads ("3 мифа о..."), carousel (до/после, рецепты), reels (упражнения), youtube (разборы), nano-banana (визуалы)

**Что работает:** До/после кейсы, развенчание мифов, простые рецепты, чек-листы.

---

## СММщик / Маркетолог / Таргетолог

**Старт:** `начать` → `customer-research` → `threads 5`

**Основные скиллы:** threads (кейсы с цифрами), offer (оффер услуг), launch (запуск курса), selling-meanings (для клиентов), swipefile (разбор кампаний)

**Что работает:** Конкретные цифры, кейсы с результатами, "закулисье работы".

---

## Онлайн-преподаватель / Эксперт по обучению

**Старт:** `начать` → `personal-unpacking` → `offer`

**Основные скиллы:** launch (запуск курса), customer-research (ЦА), offer (оффер), storytelling (прогрев), youtube (лидогенерация)

**Связка запуска:** customer-research → offer → launch → storytelling + telegram

---

## Предприниматель / Эксперт-практик

**Старт:** `начать` → `personal-unpacking` → `threads 5`

**Основные скиллы:** threads (инсайты), youtube (экспертные видео), telegram (контент + продажи), storytelling (бизнес-кейсы), selling-meanings (продающий контент)

**Что работает:** Истории из бизнеса, цифры, уроки из провалов, "как я...".

---

## Универсальный путь (для любого эксперта)

### Фаза 1: Настройка (день 1)
1. `начать` — интервью, заполнение brand/
2. Проверить первые посты: "Похожи на тебя?"
3. Если нет → `интервью` заново, показать примеры

### Фаза 2: Контент (неделя 1)
1. `threads 5` — первые посты
2. Публикация, обратная связь
3. Постепенно добавлять платформы

### Фаза 3: Глубина (неделя 2-4)
1. `распаковка личности` — уникальный голос
2. `customer-research` — понимание ЦА
3. `offer` — ядро оффера

### Фаза 4: Масштаб (месяц 2+)
1. Добавить YouTube / Reels / Карусели
2. `launch` — запуск продукта
3. `swipefile` — учиться на чужом контенте

## Сколько контента создавать

| Платформа | Минимум | Оптимум | Максимум |
|-----------|---------|---------|----------|
| Threads | 1/день | 3-5/день | 7+/день |
| Telegram | 2-3/нед | 5/нед | Ежедневно |
| YouTube | 1/нед | 2/нед | 3+/нед |
| Reels/Shorts | 2-3/нед | 5/нед | Ежедневно |
| Карусели | 1/нед | 2-3/нед | 5/нед |

**Золотое правило:** Лучше 1 пост каждый день, чем 7 постов раз в неделю.
