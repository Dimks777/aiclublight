## Связки скиллов по задачам

### "Хочу начать вести Threads"
1. **personal-unpacking** → распаковка личности
2. **customer-research** → исследование ЦА
3. **threads** → стратегия, хуки, контент-план
4. **headlines** → цепляющие первые строки
5. **copywriting** → структура текста

### "Хочу писать посты для Telegram"
1. **personal-unpacking** → если нет brand/
2. **telegram** → форматы, структуры для ТГ
3. **storytelling** → истории для вовлечения
4. **copywriting** → продающие тексты
5. **selling-meanings** → продающие смыслы

Threads → охваты. Telegram → продажи. Работают вместе.

### "Хочу запустить YouTube"
1. **youtube** → полный цикл (16 файлов!)
2. **storytelling** → сценарии с историями
3. **headlines** → заголовки видео
4. **nano-banana** → превью (thumbnails)
5. **video-editor** → субтитры, монтаж

### "Хочу делать Reels/Shorts/TikTok"
1. **reels** → ВИСП-фреймворк, 10 техник открытия
2. **storytelling** → микроистории на 60 секунд
3. **headlines** → хуки для первых 3 секунд
4. **nano-banana** → визуалы для фона
5. **heygen** → AI-аватар (если нет возможности сниматься)

Первые 3 секунды = 80% успеха.

### "Хочу делать карусели для Instagram"
1. **carousel** → структура: ХУКА → ТЕЛО → CTA
2. **nano-banana** → картинки для слайдов
3. **copywriting** → текст для слайдов
4. **headlines** → первый слайд (хук)

### "Хочу запустить продукт"
1. **customer-research** → исследование ЦА
2. **offer** → ядро оффера
3. **selling-meanings** → продающие смыслы
4. **launch** → 5 фаз запуска
5. **storytelling** → 7-актный прогрев
6. **telegram** → прогревочные посты
7. **threads** → охватные посты

Длительность: 4-8 недель.

### "Хочу разобраться в себе как эксперте"
1. **personal-unpacking** → 8 блоков (40-60 мин)
2. **customer-research** → кто мой клиент
3. **offer** → моё ядро оффера

Результат: `brand/personal-dna.md`

### "Хочу генерировать картинки"
1. **nano-banana** → промпты для Gemini
2. LaoZhang API → $0.05/картинка
3. Типы: портреты, превью YouTube, карусели, реализм

### "Хочу делать видео без съёмки"
1. **heygen** → AI-аватар из фото
2. **veo-video** → генерация из текста ($0.15-0.25/видео)
3. **nano-banana** → ключевые кадры

```
personal-unpacking → brand/personal-dna.md
         ↓
    ВСЕ контентные скиллы используют personal-dna
         ↓
threads / youtube / reels / carousel / telegram / seo-blog-writer
```

```
customer-research → карта 8 столбиков
         ↓
offer → ядро оффера
         ↓
selling-meanings → продающие смыслы
         ↓
launch → план запуска
         ↓
telegram + threads → контент для прогрева
```
