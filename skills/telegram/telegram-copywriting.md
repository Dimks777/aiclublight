## Telegram vs Threads

| Параметр | Threads | Telegram |
|----------|---------|----------|
| Формат | Ветки по 500 символов | Один пост до 4096 символов |
| Ссылки | Режутся алгоритмами | Работают без штрафа |
| Контент | Короткий, вирусный | Может быть длиннее, глубже |
| Алгоритм | Охваты зависят от engagement | Доставка всем подписчикам |
| Форматирование | Markdown ограничен | **Жирный**, *курсив*, `код`, blockquote |
| Аудитория | Холодная (лента) | Теплая (подписались сами) |

---

## Технические ограничения

| Параметр | Значение |
|----------|----------|
| Символов в посте | 4096 max |
| Форматирование | **bold**, *italic*, `code`, ~~strike~~, [ссылка](url) |
| Медиа | Фото, видео, голосовые, документы |
| Ссылки | Работают везде |
| Реакции | Стандартные emoji (настраиваемые) |

## Анатомия поста в Telegram

```
[ХУК]      -- **Жирный заголовок** = scroll-stopper
    |
[КОНТЕКСТ] -- 2-3 предложения: почему это важно
    |
[ТЕЛО]     -- Раскрытие (списки, примеры, история)
    |
[ИНСАЙТ]   -- Вывод или мораль (можно в blockquote)
    |
[CTA]      -- Мягкий призыв (опционально)
```

## Форматирование

### Жирный = scroll-stopper
В Telegram **жирный текст** выделяется сильнее всего. Используй для первой строки:
```
**Сильное утверждение.**

Дальше -- развитие мысли.
```

### Правила
- **Жирный** -- для заголовков и ключевых мыслей
- Списки с `-` или `*`
- Пустые строки между блоками
- Blockquote `>` для выводов и цитат
- Смайлики максимум 1-2 на пост, не в каждом

## CTA в Telegram

### Мягкий (основной)
- "В Фабрике показывал..."
- "В субботу на эфире разберу подробнее"
- Упоминание как факт, не продажа

### Средний
- "Пиши в личку [слово]"
- "Ссылка в описании канала"

### С вопросом
- "Как сейчас работаете с ИИ?"
- "Что думаете?"

## Голос и отличительные фразы

### Переходы
- "Короче."
- "Посмотрим."
- "К чему я это все веду:"
- "Смотрите."

### Усилители
- "Вот я за что топлю:"
- "Нифига не легко."
- "Я бросал."

### Завершения
- "Что дальше?"
- "Как у вас с этим?"
- "Делитесь в комментах."

**Важно:** Это пример голоса Max. Для персонализации используй скилл tone-of-voice (если настроен для проекта).

## ТГ прогрев при запуске

ТГ = место где расширяется глубина прогрева. Дает +10-50% выручки. Работает параллельно с Инстой.

### Правила дублирования
- **50% смыслов** из Инсты, но в ДРУГОМ формате и с другого захода
- Менять формат каждые 1-2 поста
- Разные заходы к одним и тем же смыслам

### Структура закрепа

1. Заголовок (цепляющий)
2. Самопрезентация с позиционированием + личные кейсы
3. Чем отличаюсь от конкурентов
4. О чем канал, чем полезен
5. Навигация по лучшим постам
6. Упоминание продукта + кейсы
7. Призыв подписаться

### Форматы для прогрева

| Формат | Частота | Эффект |
|--------|---------|--------|
| Экспертные посты на боли | Каждый день | Основа прогрева |
| Голосовые подкасты | 10+ за прогрев | Высокая лояльность |
| Разборы/эфиры | 1/нед | 15-20% выручки |
| Кейсы (развернутые) | 2-3/нед | Социальное доказательство |
| Лайф/инфоповоды | По ситуации | Поддержка охватов |
| Длинный подкаст про продукт | 1 шт | Killer feature запуска |

## Telegram Anti-patterns (платформенные)

| Ошибка | Как исправить |
|--------|---------------|
| Копипаст из Инсты | Другой формат, другой заход |
| Пост без форматирования | Жирный для ключевых мыслей, абзацы |
| Слишком длинный без структуры | Разбить на блоки, добавить списки |
| Слишком много смайликов | Максимум 1-2 на пост |

Универсальные антипаттерны текста: см. text-posts (post-anatomy.md).

## Правила от Gary Vaynerchuk

Для теплой аудитории (>3 мес) пропорция пользы к продажам может быть 30/70. Теплым нужен пинок, не ещё одна порция пользы. Один смысл -> разные форматы под платформу. НЕ копипаст между Инстой и ТГ.
