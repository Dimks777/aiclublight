## Telegram vs Threads хуки

| Аспект | Threads | Telegram |
|--------|---------|----------|
| Видимость | Первая строка в ленте | Начало поста в превью |
| Длина | До 100-150 символов | Можно длиннее |
| Формат | Plain text | **Жирный** = scroll-stopper |
| Эмоции | Обязательны | Можно спокойнее (лояльная аудитория) |

---

## Telegram-специфичные правила

### Жирный = власть
В Telegram **жирный текст** выделяется сильнее всего. Первая строка -- всегда жирная:
```
**Сильное утверждение.**

Дальше -- развитие мысли.
```

### Можно длиннее
В Telegram подписчики лояльнее -- можно 2-3 предложения в хуке:
```
**Залетевший ролик -- это не везение.**

Это статистика. Когда выпускаешь 30 роликов -- математика начинает работать.
```

### Меньше кликбейта
Telegram-аудитория = подписчики, которые выбрали тебя. Работает:
- Честность
- Конкретика
- Личный опыт

## Telegram-специфичные формулы хуков

### Жирный заголовок-утверждение
```
**[Сильное утверждение с конкретикой].**
```
- **На моей фабрике контента сотрудник No1 -- ИИ-сценарист.**
- **Залетевший ролик -- это не везение. Это статистика.**
- **Реалистичные AI-фото -- тренд 2026?**

### Хук с числами
```
**[Число] + [контекст]**
```
- **+151 подписчик в телеграм за 13 дней**
- **100 км за неделю** -- и вот что я понял
- **$4 за минуту** видео с аватаром

### Результат + Эмоция
```
**[Результат]!** [Короткий контекст]
```
- **Первая продажа с Telegram!** 3 недели работы.
- **Ученик сделал первые 100k** -- разбираю, что сработало.

## Telegram Anti-patterns хуков

| Плохой хук | Как исправить |
|------------|---------------|
| **Кайф.** (одно слово) | Добавить конкретику: "Кайф, когда план работает" |
| Без жирного | Первая строка -- **жирная** |
| Определение вместо результата | "ИИ -- это технология..." -> "Сценарий за 30 минут вместо 4 часов" |

Универсальные антипаттерны: см. text-posts (hooks-and-cta.md).
