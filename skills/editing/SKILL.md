## Режим работы

## Главные правила инфостиля (Ильяхов)

### 1. Без воды
- Убирай слова, которые не несут смысла
- Каждое предложение должно сообщать новое
- Если абзац можно удалить без потери смысла — удаляй

### 2. Конкретика вместо оценок
- ❌ «Мы — лидеры рынка с большим опытом»
- ✅ «Работаем с 2015 года, 340 проектов в портфолио»

### 3. Факты вместо мнений
- ❌ «Отличный результат»
- ✅ «Конверсия выросла с 1,2% до 3,8% за 2 месяца»

### 4. Простой синтаксис
- Одна мысль — одно предложение
- Подлежащее → сказуемое → дополнение
- Без причастных оборотов на 3 строки

### 5. Структура
- Заголовок отвечает на вопрос читателя
- Первый абзац — главное (перевёрнутая пирамида)
- Подзаголовки как навигация: по ним можно понять весь текст

## Правила убеждающего текста (Панда)

### 1. Фокус на читателе
- Текст о нём, не о тебе
- ❌ «Мы разработали уникальную методику»
- ✅ «Вы освоите методику за 2 недели и начнёте применять»

### 2. Конкретная выгода
- Не свойство, а результат для читателя
- ❌ «Курс из 12 модулей»
- ✅ «После 12 модулей вы сможете вести проекты самостоятельно»

### 3. Доказательства
- Цифры, кейсы, отзывы, скриншоты
- Социальное доказательство: «340 выпускников»
- Авторитет: регалии, публикации, клиенты

### 4. Работа с сомнениями
- Предугадать возражение и ответить в тексте
- Гарантии снижают риск: «Вернём деньги за 14 дней»

### 5. Призыв к действию
- Один текст — один CTA
- Глагол + результат: «Записаться и получить программу»
- Без давления: объяснить, что произойдёт после нажатия

## Чек-лист редактуры (применяй последовательно)

### Проход 1 — Смысл
- [ ] Понятно, о чём текст, за 5 секунд?
- [ ] Есть ли главная мысль? Она в начале?
- [ ] Каждый абзац двигает к цели текста?
- [ ] Нет повторов и пересказа одного и того же?

### Проход 2 — Инфостиль
- [ ] Убраны стоп-слова (см. references/ilyahov-rules.md)
- [ ] Оценки заменены на факты
- [ ] Нет канцеляризмов и штампов
- [ ] Нет ИИ-конструкций: «это не про X, это про Y», «и нет, это не про…»
- [ ] Предложения короткие, синтаксис простой

### Проход 3 — Убеждение (для продающих текстов)
- [ ] Текст о читателе, не о компании
- [ ] Свойства переведены в выгоды
- [ ] Есть доказательства (цифры, кейсы)
- [ ] Возражения отработаны
- [ ] CTA конкретный и один

### Проход 4 — Анти-AI фильтр (убрать следы генерации)

Основано на Wikipedia "Signs of AI writing" + русскоязычные паттерны.

**Лексика — стоп-лист AI-слов (убрать или заменить):**
- уникальный, инновационный, революционный, прорывной -> конкретика: что именно?
- комплексный подход, эффективное решение -> убрать, слишком абстрактно
- ключевой, важнейший, значимый -> убрать или доказать фактом
- осуществлять -> делать
- обеспечивать -> давать
- являться -> быть (или убрать)
- предоставлять возможность -> чтобы можно было
- способствовать -> помогать
- погрузиться (в тему) -> разобраться / покопаться
- на сегодняшний день -> сейчас / сегодня
- в рамках -> внутри / по ходу
- безусловно -> да / конечно
- следовательно -> значит / поэтому
- кроме того -> а ещё / плюс

**Вводные-паразиты (убрать полностью):**
- Важно отметить, что...
- Стоит подчеркнуть...
- Не секрет, что...
- В современном мире...
- В условиях цифровизации/конкуренции...
- Давайте углубимся...

**Конструкции-маркеры (переписать):**
- "Не просто X, а Y" / "Не только X, но и Y" -> переформулировать
- "Во-первых... Во-вторых... В-третьих..." -> убрать нумерацию, сделать текст
- "С одной стороны... с другой..." -> показать конфликт живым примером
- "Таким образом" / "В заключение" / "Подводя итог" -> просто закончить мысль
- "Эксперты отмечают" / "Исследования показывают" -> кто конкретно? какое исследование?

**Деепричастные нагромождения (разбить на простые предложения):**
- ДО: "Являясь ключевым фактором, обеспечивая коммуникацию, предоставляя отчёты..."
- ПОСЛЕ: "Команда наладила связь и сдавала отчёты. Всё прошло гладко."
- Правило: максимум 1 деепричастие на предложение, лучше 0

**Структурные маркеры:**
- [ ] Предложения РАЗНОЙ длины? (чередуй короткие и длинные)
- [ ] Абзацы РАЗНОЙ длины? (не все по 3-4 строки)
- [ ] Нет правила трёх? (AI любит тройки: "скорость, качество, надёжность")
- [ ] Нет монотонных списков с жирными заголовками?
- [ ] Нет идеально параллельных конструкций?
- [ ] Начало -- не "В современном мире" / "Вы когда-нибудь задумывались"?
- [ ] Конец -- не общие слова про "будущее" / "перспективы"?

**Стиль для соцсетей (добавить жизнь):**
- [ ] Есть "я" / личный опыт / мнение?
- [ ] Есть разговорные обороты (уместные)?
- [ ] Есть ирония, юмор, провокация?
- [ ] Есть конкретика (цифры, имена) вместо абстракций?
- [ ] Эмоции ВЫЗЫВАЮТСЯ примером, а не НАЗЫВАЮТСЯ ("это круто")?
- [ ] Текст звучит как живой человек, а не Wikipedia?

**Двухпроходная проверка (финальный аудит):**
1. Прочитай текст и спроси: "Что делает этот текст похожим на AI?"
2. Запиши 2-3 оставшихся маркера
3. Перепиши эти места
4. Текст должен пройти "тест бара": можно ли это сказать другу вслух?

### Проход 5 — Финал
- [ ] Заголовок цепляет и отражает содержание
- [ ] Текст читается вслух без запинок
- [ ] Форматирование помогает сканированию
- [ ] Длина адекватна задаче (не раздут, не обрезан)
