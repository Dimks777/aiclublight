## Золотое правило

> **«Всё, что может быть не прочитано — будет не прочитано»**

Задача каждого предложения — заставить прочитать следующее. Это и есть скользкая горка.

## 18 правил сильного текста (чек-лист)

### Обязательные
1. **У каждого текста — конкретная цель.** «Что текст даст читателю?» + «Что даст мне?»
2. **Пиши со скальпелем.** Сократи в 1.5–2 раза. Нет лишних тезисов, примеров, оборотов
3. **Пиши пьяным, редактируй трезвым.** Не редактируй в процессе написания
4. **Будь за белых или за красных.** Никаких «можно посмотреть по-разному». Конкретная позиция
5. **Не дави на клиента.** Максимум 3 коммерческих слова на текст. Без агрессивных призывов
6. **Текст посвящён читателю.** Через призму «а что это даст моему читателю?»
7. **Максимальная конкретика.** Цифры, факты, имена, достижения. Мозг фильтрует абстракции
8. **Избегай канцеляризмов.** «Зелёный массив» → «лес». Тёплый человечный язык
9. **Выключи писателя, включи художника.** Визуальные образы, вкус, запах, текстура
10. **Следи за активностью.** Читатели воспринимают текст вместе с комментариями
11. **Всё что может быть неправильно понято — будет.** Вылавливай двусмысленности
12. **Упрощай конструкции.** Разбивай сложные предложения на простые
13. **Проговаривай вслух.** Уши услышат то, что не увидят глаза
14. **Избегай клише.** «Уютный уголок», «индивидуальный подход» — мёртвые слова
15. **Запрещена конструкция «это не про X, это про Y».** Маркер ИИ-текста. Писать прямо, без шаблонных противопоставлений

### Усиливающие
16. **Пиши не только чтобы прочитали, но чтобы пересказали.** Факт про 2500 хинкали > «опытный повар»
17. **Продающий текст = язык задач клиента.** Не «купите», а «как решить вашу проблему»
18. **Экологичность.** Без обмана, без манипуляций, без фейковых отзывов

## Структура поста (6 блоков)

```
1. КОНТЕКСТ — актуализация задачи/проблемы
2. КЛЮЧЕВОЕ СООБЩЕНИЕ — что хотим донести
3. АРГУМЕНТАЦИЯ — факты, цифры, кейсы
4. ДЕТАЛИ — от общего к частному
5. РЕЗЮМЕ — повтор главной мысли
6. CTA — призыв к действию
```

## Формула продающего текста

> «Продавать не продавая» — главный лозунг

```
Подчеркни важность задачи/проблемы
→ Свяжи задачи клиента с продуктом
→ Подкрепи фактами, кейсами, цифрами
→ Дай комфортную точку входа
```

**Запрещено:** слова «цена», «акция», «купить», «заказать»
**Вместо них:** «сэкономить», «встроить в систему», «обеспечить себя»

## Распределение контента

### Для компании:
| Тип | Доля | Что |
|-----|------|-----|
| Промо | 40–50% | Обзоры, тесты, гиды, шаблоны |
| Познавательный | 20% | Мастер-классы, лайфхаки |
| Пользовательский | 10–15% | Отзывы, кейсы, Q&A |
| Развлекательный | 10–15% | Юмор, неформал (без «котиков») |
| Вовлекающий | 10–15% | Опросы, викторины, дискуссии |

### Для персонального бренда:
| Тип | Доля |
|-----|------|
| Экспертный | 40% |
| Продающий | 40% |
| Лайфстайл | 20% |

## Работа с ценой

| Правило | Как |
|---------|-----|
| Не в начале | Сначала ценность, потом цена |
| Обоснуй | Что входит, что сэкономит |
| Покажи дорогие альтернативы | Дорогой пакет → средний кажется нормальным |
| Прозрачность | Анатомия цены: бюджет + ФОТ + прибыль |
| Сравни с рынком | Средние цены из независимых источников |
| Премиум → без цены | «Связаться с менеджером» |

## Работа с возражениями

### 2 подхода:
1. **Недирективный** — показываем сильные стороны через кейсы, репортажи, отзывы
2. **Прямой** — озвучиваем возражение и тут же снимаем

### Инструменты снятия:
- Отзывы клиентов
- Авторитетные рекомендации
- Цифры и соц. доказательство
- Кейсы со ссылками
- Бэкстейдж производства
- Тест продукта в прямом эфире
- «Хотите — проверьте»

### Формулировки входа:
```
«Возможно, у вас возникла мысль…»
«Кто-то может подумать, что…»
«Существует мнение, что…»
```

**Не усиливай возражение!** Без «Очень часто мы слышим от клиентов…»

## Призыв к действию (CTA)

### 3 типа:
| Тип | Примеры |
|-----|---------|
| Коммуникационный | Лайк, комментарий, репост, сохранение |
| Продающий | Ссылка, заявка, консультация, регистрация |
| Эмоциональный | Запомните, улыбнитесь, начните действовать |

### Формула контекстного CTA:
```
Действие + мотив для читателя
«Сохраните пост, чтобы сервисы были под рукой»
«Поставьте лайк, чтобы чаще видеть мои посты»
```

**Максимум 5 CTA на текст. Не повторять один и тот же.**

## Связка "Боль + Кейс"

### Основная формула контента при запуске

```
БОЛЬ/ПОТРЕБНОСТЬ/ВОЗРАЖЕНИЕ + ЗАКРЫТИЕ КЕЙСОМ/ОТСЫЛКОЙ К ПРОДУКТУ
```

Каждый день прогрева:
1. Поднять боль (из карты смыслов / кастдевов / Директа)
2. Раскрыть через сторителлинг/экспертность
3. Закрыть кейсом ученика или отсылкой к продукту

## Правила мировых экспертов

### Donald Miller (StoryBrand)
- Эксперт = Guide (проводник), клиент = Hero
- One-Liner: "[ЦА] застряла в [проблема]. [Продукт] даёт [план]. Результат: [трансформация]"

### Sabri Suby (Result > Process)
- Продавать РЕЗУЛЬТАТ, не программу. 1 тезис = 1 боль + конечный результат

### Dan Kennedy (Direct Response)
- Каждый контакт заканчивается конкретным следующим шагом. Нет CTA = потерянный контакт
